# Bookies

A web application for managing book clubs.

## Features (planned)

- User accounts
- Create/join book clubs
- Monthly book tracking
- Book suggestions
- Voting system

## Tech Stack

Backend:
- FastAPI
- Python

Frontend:
- React
- TypeScript

Database:
- SQLite (development)
- PostgreSQL (production)
