# Bookies

A web application for managing book clubs.

## Features (curret)
- FastAPI backend
- SQLite database
- User registration
- Secure password hashing
- User authentication
- Create/join book clubs
- Monthly reading tracking
- Book suggestions
- Voting and Suggestion systems
- Private/Public Book Clubs
- Reading Status
- Book rating/reviews
- Google OAuth login
- Reading history
- Club and User Dashboard

## Features (planned)
- Book discussions
- Private Club Invites

## Tech Stack

Backend:
- FastAPI
- Python
- SQLAlchemy
- SQLite

Frontend:
- React Router (planned)   
- TypeScript

Database:
- SQLite (development)
- PostgreSQL (production)
