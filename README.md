# Bookies

A web application for managing book clubs.

## Features (curret)
- FastAPI backend
- SQLite database
- User registration
- Secure password hashing

## Features (planned)
- User authentication
- Create/join book clubs
- Monthly reading tracking
- Book suggestions
- Voting system
- Google OAuth login
- Reading history
- Book discussions

## Tech Stack

Backend:
- FastAPI
- Python
- SQLAlchemy
- SQLite

Frontend:
- React (planned)   
- TypeScript

Database:
- SQLite (development)
- PostgreSQL (production)
